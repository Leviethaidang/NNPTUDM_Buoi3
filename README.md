# NNPTUDM_Buoi3

Products Dashboard

Open [index.html](index.html) in a browser to use the dashboard. It fetches products from https://api.escuelajs.co/api/v1/products and provides search, sort, pagination, CSV export, create and edit features.

Notes:
- Create and edit use the fake API endpoints; persistence depends on the API behavior.
- Use modern browsers for full functionality.
